## ğŸš€ Quick Start

### Prerequisites
- Modern web browser
- Internet connection (for CDN resources)
- Optional: Node.js for http-server

### Installation & Setup

1. **Clone or Download the portfolio**
   ```bash
   git clone <repository-url>
   cd portfolio
   ```

2. **Install http-server globally (Optional)**
   ```bash
   npm install -g http-server
   ```

3. **Start the development server**
   ```bash
   http-server -p 8000
   ```

4. **Open in browser**
   ```
   http://localhost:8000
   ```

### Alternative Setup (Python)
```bash
# Using Python's built-in server
python -m http.server 8000
```
