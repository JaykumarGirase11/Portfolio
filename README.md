## 🎯 Features

✅ Clean & Modern UI/UX with Gradient Effects  
✅ Animated Typing Text with Role Switcher 🔠  
✅ tsParticles Background Animation (Stars + Lines) ✨  
✅ Interactive Flip Cards for Projects, Education, and Skills
